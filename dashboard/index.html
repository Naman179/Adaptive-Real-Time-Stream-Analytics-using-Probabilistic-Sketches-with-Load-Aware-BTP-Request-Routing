Count-Min Sketch
d×w counter array. Frequency estimated as min over d hash positions.
f̂(x) ≤ f(x) + ε·N, prob 1−δ
w=⌈e/ε⌉  ·  d=⌈ln 1/δ⌉  ·  O(wd)
HyperLogLog
m=2p registers. Estimates cardinality via harmonic mean of leading-zero runs.
Rel. error ≈ 1.04 / √m
p=12 → 4096 regs → ±1.6%  ·  O(m)
Misra–Gries
Keeps k−1 candidates. Items surviving appear > N/k times. Sliding window eviction.
Error ≤ N / k per candidate
k candidates  ·  window=10,000  ·  O(k)
Adaptive Controller
Rule-based loop every 5 s. Trades accuracy for latency/memory by switching templates.
min α·Lat + β·Mem + γ·Err
P99>200ms → downgrade  ·  err>5% → upgrade
